A simple way to manage browser bookmarks

Bookmark Manager is a web app that makes it easy to save and manage your browser bookmarks. You can save and retrieve your bookmarks from anywhere.

How it works

  1. Create an account using the form below
  2. Upon registration you will get 2 buttons for saving and retrieving bookmarks.
  3. Drag the buttons to your bookmarks toolbar and start using Bookmark Manager
